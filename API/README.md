# ComfyUI云平台 API中心

一个现代化的API调用页面首页，专为ComfyUI云平台设计，旨在降低用户认知成本，快速匹配用户需求与对应API类型。

## 🎯 核心目标

- **降低用户认知成本**：通过清晰的视觉设计和直观的分类帮助用户快速理解平台能力
- **快速匹配需求**：通过场景化引导帮助用户找到最适合的API类型
- **引导用户进入详情页**：提供便捷的入口和行动按钮

## ✨ 核心特性

### 1. 清晰的价值主张
- 突出平台"从基础模型到定制工作流"的全链路API能力
- 明确传达"无需部署算力，直接调用"的核心优势

### 2. 三大API分类
- **标准模型API**：调用预训练模型的基础能力，即调即用
- **AI应用API**：调用现成的AI应用，直接复用完整功能  
- **ComfyUI工作流API**：调用自定义或社区的ComfyUI工作流

### 3. 场景化决策指南
- 通过"我该选哪种API？"的引导式设计
- 用具体场景帮助用户快速做出选择
- 点击式交互，实时高亮对应API卡片

### 4. 现代化UI设计
- 响应式设计，支持桌面端和移动端
- 流畅的动画效果和悬停交互
- 清晰的颜色区分和视觉层次

## 🚀 快速开始

1. 直接在浏览器中打开 `index.html` 文件
2. 或者使用本地服务器：
   ```bash
   # 使用Python
   python -m http.server 8000
   
   # 使用Node.js
   npx serve .
   ```

## 🎨 设计亮点

### 视觉设计
- **渐变背景**：使用紫色渐变营造科技感
- **卡片设计**：圆角卡片配合阴影效果，现代简洁
- **颜色系统**：
  - 标准模型：蓝色系（稳定、基础）
  - AI应用：绿色系（高效、即用）
  - ComfyUI工作流：紫色系（灵活、定制）

### 交互设计
- **悬停效果**：卡片悬停时轻微放大和阴影加深
- **点击反馈**：按钮点击有视觉反馈和通知提示
- **滚动动画**：元素进入视口时的渐入动画
- **决策引导**：点击决策指南可高亮对应API卡片

### 响应式设计
- **桌面端**：三列网格布局，充分利用屏幕空间
- **平板端**：自适应网格，保持良好的视觉比例
- **移动端**：单列布局，优化触摸交互

## 🔧 技术特性

- **纯前端实现**：无需后端，可直接部署到静态托管
- **现代CSS**：使用Flexbox和Grid布局，CSS变量和动画
- **原生JavaScript**：无依赖，轻量级交互实现
- **性能优化**：防抖函数、Intersection Observer API
- **无障碍支持**：键盘导航和屏幕阅读器友好
